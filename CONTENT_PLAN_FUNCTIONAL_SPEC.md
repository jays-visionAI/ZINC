# 콘텐츠 플랜 기능 명세서 (Content Plan Functional Specification)
**Version 1.1 (Korean)**
본 문서는 Knowledge Hub의 각 콘텐츠 플랜 유형별로 특화된 UI/UX 및 기술적 기능을 정의합니다. 정적인 텍스트 결과물을 넘어, 사용자가 상호작용하고 실제 업무에 활용 가능한 도구로 전환하는 것을 목표로 합니다.

---

## 1. STRATEGIC PLANS (전략 기획)
*목표: 장기적인 계획 수립, 구조 정의, 워크플로우 관리*

### 1.1. Campaign Brief (캠페인 브리프)
*   **핵심 가치**: 캠페인 전체를 총괄하는 중앙 관제 문서.
*   **UI/UX 컨셉**: **"위키 스타일 대시보드"**
    *   좌측 사이드바 내비게이션 (목표, 타겟, 핵심 메시지, 채널 믹스 등 섹션 이동).
    *   섹션별 "상태" 배지 수정 기능 (초안, 승인됨, 진행 중).
*   **킬러 기능 (Killer Features)**:
    *   **팀 코멘트 (Comments)**: 브리프 내용 옆에 메모나 피드백을 남길 수 있는 기능.
    *   **태스크 생성 (Create Tasks)**: 브리프의 실행 계획(Execution Plan)을 칸반 보드 카드나 CSV(Jira용)로 자동 변환.

### 1.2. Content Calendar (콘텐츠 캘린더)
*   **핵심 가치**: 콘텐츠 배포 일정 시각화 및 관리.
*   **UI/UX 컨셉**: **"인터랙티브 캘린더 그리드"**
    *   월간/주간 전체 보기 뷰.
    *   플랫폼별 색상 코딩 (트위터=파랑, 인스타=핑크, 블로그=초록).
*   **킬러 기능**:
    *   **드래그 앤 드롭**: 게시물 카드를 끌어서 날짜 변경.
    *   **캘린더 내보내기**: 구글 캘린더/아웃룩용 `.ics` 파일 다운로드.
    *   **동기화 상태**: "예약됨(Scheduled)" vs "작성 중(Draft)" 토글 스위치.

### 1.3. Channel Strategy (채널 전략)
*   **핵심 가치**: 채널별 최적화 전략 및 규칙 정의.
*   **UI/UX 컨셉**: **"플랫폼 비교 매트릭스"**
    *   열(Column): 각 채널 (인스타그램, 링크드인, 블로그).
    *   행(Row): 콘텐츠 유형, 업로드 주기, 어조(Tone), KPI.
*   **킬러 기능**:
    *   **골든 타임 시각화**: 채널별 추천 업로드 시간을 히트맵 차트로 표시.
    *   **KPI 계산기**: 현재 팔로워 수를 입력하면 목표 도달률(Engagement) 자동 계산.

### 1.4. Brand Positioning & Messaging (브랜드 포지셔닝)
*   **핵심 가치**: 브랜드의 정체성과 목소리(Voice) 정의.
*   **UI/UX 컨셉**: **"인터랙티브 브랜드 피라미드"**
    *   시각적인 피라미드 다이어그램 (본질 -> 가치 -> 성격 -> 속성).
    *   각 층을 클릭하면 상세 설명이 펼쳐짐.
*   **킬러 기능**:
    *   **보이스 시뮬레이터 (Voice Simulator)**: 사용자가 아무 말이나 입력하면, 브랜드 톤앤매너로 재작성해주는 테스트 창.
    *   **경쟁사 포지셔닝 맵**: 2차원 사분면 차트에 우리 브랜드와 경쟁사 위치 표시 (예: Luxury vs Value).

## 2. QUICK ACTIONS (빠른 실행)
*목표: 즉각적인 실행, 복사/붙여넣기 편의성, 빠른 아이디어 도출*

### 2.1. Social Post Ideas & Trend Response (소셜 포스팅)
*   **핵심 가치**: 피드에 올릴 준비가 된 콘텐츠 생성.
*   **UI/UX 컨셉**: **"피드 에뮬레이터 (Feed Emulator)"**
    *   실제 인스타그램/트위터/링크드인 피드와 똑같은 디자인.
    *   프로필 사진, 핸들, 캡션, 이미지 영역 표시.
*   **킬러 기능**:
    *   **원클릭 이미지 생성**: "이 포스트용 이미지 만들기" 버튼 (AI 이미지 생성 연동).
    *   **해시태그 선택기**: 추천된 해시태그를 클릭하여 포함/제외 토글.
    *   **복사하기**: 텍스트와 해시태그를 포맷에 맞춰 클립보드에 복사.

### 2.2. Ad Copy Variants (광고 카피)
*   **핵심 가치**: 다양한 후킹(Hook) 메시지 테스트.
*   **UI/UX 컨셉**: **"A/B 테스트 카드"**
    *   "Variant A"와 "Variant B"를 나란히 비교하는 카드 뷰.
*   **킬러 기능**:
    *   **예측 점수 (Predictive Score)**: 어떤 카피가 더 성과가 좋을지 AI가 점수 매기기 (간결성, 감정 단어 기반).
    *   **목업 모드 (Mockup Mode)**: "텍스트만 보기" vs "페이스북 광고 미리보기" 전환.

### 2.3. Action Items (액션 아이템)
*   **핵심 가치**: 분석 내용을 할 일 목록으로 변환.
*   **UI/UX 컨셉**: **"체크리스트 앱"**
    *   심플한 할 일 목록 UI.
    *   우선순위 깃발 표시 (High/Medium/Low).
*   **킬러 기능**:
    *   **진행률 바**: 항목을 체크할 때마다 진행률 게이지가 차오름.
    *   **위임하기 (Delegate)**: 특정 항목을 에이전트(예: 카피라이터 봇)에게 할당하여 바로 작업 시작.
    *   **노션/슬랙 내보내기**: 리스트를 서드파티 툴 포맷으로 복사.

## 3. KNOWLEDGE PLANS (지식 분석)
*목표: 심층 분석, 생태계 이해, 시각적 데이터*

### 3.1. Audience Persona (타겟 페르소나)
*   **핵심 가치**: 타겟 고객을 구체적 인물로 형상화.
*   **UI/UX 컨셉**: **"RPG 캐릭터 카드"**
    *   페르소나의 얼굴 사진 (AI 생성).
    *   능력치 바 (기술 친숙도, 구매력, 충성도 등).
*   **킬러 기능**:
    *   **페르소나와 대화하기 (Chat with Persona)**: 채팅창에서 질문하면 AI가 해당 페르소나에 빙의하여 답변 ("우리 가격 정책 어때?" -> "좀 비싼 것 같은데요?").
    *   **하루 일과표 (Day in the Life)**: 페르소나의 하루 루틴을 시각적 타임라인으로 표시.

### 3.2. Competitor Analysis (경쟁사 분석)
*   **핵심 가치**: SWOT 및 비교 우위 파악.
*   **UI/UX 컨셉**: **"배틀 카드 (Battle Card)"**
    *   자사 vs 경쟁사를 1:1로 비교하는 컬럼 뷰.
*   **킬러 기능**:
    *   **SWOT 매트릭스**: 2x2 인터랙티브 그리드 (강점, 약점, 기회, 위협).
    *   **실시간 데이터 갱신**: "웹사이트 다시 확인" 버튼을 눌러 최신 정보 반영 (추후 에이전트 연동).

### 3.3. Brand Mind Map (브랜드 마인드맵)
*   **핵심 가치**: 연관 개념 시각화.
*   **UI/UX 컨셉**: **"인터랙티브 노드 그래프"**
    *   둥둥 떠다니는 키워드 버블 (Force-directed Graph).
*   **킬러 기능**:
    *   **무한 확장**: 키워드 버블을 클릭하면 하위 키워드가 펑 하고 튀어나옴.
    *   **클러스터링**: 연관된 주제끼리 같은 색상으로 그룹화.

## 4. CREATE NOW (제작 도구)
*목표: 최종 산출물 제작, 포맷팅 된 문서, 디자인*

### 4.1. Product Brochure & One-Pager & Press Release (브로슈어/보도자료)
*   **핵심 가치**: 인쇄하거나 배포 가능한 완성형 문서.
*   **UI/UX 컨셉**: **"WYSIWYG 문서 에디터"**
    *   A4 용지 비율의 캔버스 (흰 배경, 그림자 효과).
    *   실제 문서 같은 타이포그래피 (헤드라인, 본문).
*   **킬러 기능**:
    *   **PDF 다운로드**: 작성된 문서를 그대로 PDF로 저장.
    *   **이미지 드래그 앤 드롭**: 생성된 이미지를 문서 내 원하는 곳에 배치.
    *   **테마 스위처**: "기업용 블루 스타일" vs "모던 블랙 스타일" 등 디자인 테마 즉시 변경.

### 4.2. Pitch Deck Outline (피치덱 아웃라인)
*   **핵심 가치**: 프레젠테이션 스토리라인 구성.
*   **UI/UX 컨셉**: **"슬라이드 소터 (Slide Sorter)"**
    *   파워포인트의 '여러 슬라이드 보기' 처럼 네모난 슬라이드 썸네일 그리드.
*   **킬러 기능**:
    *   **PPT 내보내기**: 기본 `.pptx` 파일로 변환하여 다운로드.
    *   **발표자 노트 보기**: 슬라이드 그림 아래에 스크립트(대본) 표시/숨김.

### 4.3. Email Template (이메일 템플릿)
*   **핵심 가치**: HTML 이메일 구조.
*   **UI/UX 컨셉**: **"이메일 클라이언트 미리보기"**
    *   보낸 사람, 받는 사람, 제목, 본문이 실제 메일함처럼 보임.
    *   모바일/데스크톱 뷰 토글.
*   **킬러 기능**:
    *   **나에게 테스트 발송**: 버튼 한 번으로 내 메일로 테스트 전송.
    *   **HTML 코드 복사**: 메일침프(Mailchimp) 등 마케팅 툴에 붙여넣을 수 있는 HTML 소스 복사.

## 5. 구현 로드맵 (우선순위 제안)
1.  **Phase 1 (즉각적 가치)**: 소셜 피드 미리보기, PDF 내보내기, 페르소나 대화하기.
2.  **Phase 2 (시각화)**: 마인드맵 그래프, 캘린더 드래그 앤 드롭, 광고 목업.
3.  **Phase 3 (연동)**: 캘린더/PPT 내보내기, 태스크 툴(Jira/Slack) 연동.
