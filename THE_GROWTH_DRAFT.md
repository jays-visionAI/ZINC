#### ROI 계산 공식

```javascript
// ROI 계산
const roi = {
  // 투입
  input: {
    zynkCost: subscription.monthlyFee,        // $127
    setupTime: 2,                              // 시간
    setupTimeCost: 2 * hourlyRate              // $100
  },
  
  // 산출
  output: {
    contentCount: 89,                          // 발행 콘텐츠
    manualTimePerContent: 0.5,                 // 콘텐츠당 30분 (수동 시)
    savedTime: 89 * 0.5,                       // 44.5시간
    savedTimeCost: 44.5 * hourlyRate,          // $2,225
    
    reachIncrease: 23,                         // %
    followerGrowth: 2340,
    engagementRate: 5.8
  },
  
  // ROI = (Output - Input) / Input * 100
  roiPercent: ((2225 - 227) / 227) * 100       // ≈ 880%
}
```

## 🔗 다른 모듈과의 연동

### 데이터 흐름

| From | To | 데이터 |
|------|-----|--------|
| **The Filter** | **The Growth** | 발행된 콘텐츠 ID, 발행 시간 |
| **소셜 플랫폼 API** | **The Growth** | 도달, 참여, 팔로워 데이터 |
| **The Growth** | **Brand Brain** | 학습된 패턴 (자동 반영) |
| **The Growth** | **Studio** | 최적 발행 시간 (자동 반영) |
| **The Growth** | **The Filter** | 효과적인 해시태그 (자동 반영) |

### Brand Brain 자동 반영 예시

```javascript
// The Growth에서 학습한 패턴

  optimalPostingTime: { day: "화요일", time: "09:00" },

  contentLength: { min: 150, max: 200 },
  imageStyle: "밝은 톤 + 자연 배경"
}

// Brand Brain에 자동 반영
brandBrain.autoUpdate({
  strategy: {
    currentFocus: {
      keywords: [...existing, "#클린뷰티", "#비건화장품"]
    }
  },
  learningApplied: true,
  appliedAt: Timestamp
})
```
