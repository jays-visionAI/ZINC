## 📚 LAYER 1: Knowledge Hub

### 목적
**"고객으로부터 데이터를 수집하고, 전략 수립을 위한 기초 데이터를 생성"**

### 1.1 입력 (Data Collection)

| 데이터 소스 | 설명 | 수집 방식 |
|------------|------|----------|
| **고객 제공 문서** | 회사소개서, IR자료, 브랜드가이드 등 | Google Drive 연동 |
| **고객 제공 링크** | 공식 웹사이트, YouTube 채널 | URL 입력 |
| **고객 제공 노트** | 비공식 정보, 내부 메모 | 직접 입력 |
| **외부 검색 데이터** | 고객 정보 기반 관련 데이터 | AI 자동 검색 |

### 1.2 출력 (Foundational Data)

Knowledge Hub가 생성하여 **Strategy War Room에 전달하는** 기초 데이터:

| 기초 데이터 | 설명 | 생성 방법 |
|------------|------|----------|
| **브랜드 프로필** | 회사 개요, 미션, 비전, 핵심 가치 | 문서 분석 |
| **제품/서비스 맵** | 제품 라인업, 특징, 가격대 | 카탈로그 분석 |
| **타겟 고객 프로필** | 데모그래픽, 관심사, 행동 패턴 | IR자료 + AI 추론 |
| **경쟁사 프로필** | 경쟁사 목록, 강점/약점, 포지셔닝 | 외부 검색 |
| **시장 컨텍스트** | 산업 트렌드, 규제, 기회 요인 | 외부 검색 |

## 🎯 LAYER 2: Strategy War Room

### 목적
**"6가지 브랜드 가치 지표를 모니터링하고, 브랜드 자산 성장을 위한 전략을 AI가 추천 및 실행 계획 수립"**

### 2.1 6가지 브랜드 가치 지표 (Brand Value Indicators)

| # | 지표 | 설명 | 측정 방법 |
|---|------|------|----------|
| 1 | **Brand Awareness** | 브랜드 인지도 | 멘션 수, 검색량, 도달률 |
| 2 | **Brand Perception** | 브랜드 인식 | 감성 분석, 리뷰 점수 |
| 3 | **Brand Engagement** | 고객 참여도 | 좋아요, 댓글, 공유, CTR |
| 4 | **Brand Loyalty** | 고객 충성도 | 재방문율, 팔로워 유지율 |
| 5 | **Brand Differentiation** | 차별화 | 경쟁사 대비 포지셔닝 |
| 6 | **Brand Consistency** | 일관성 | 채널별 메시지 일관성 |
