## 1. Executive Summary

### 1.1 목적
고객의 브랜드 자산을 수집하고 AI가 분석하여 **마케팅/콘텐츠 전략 기획안**을 생성하는 시스템

### 1.2 핵심 가치
| 가치 | 설명 |
|------|------|
| 데이터 수집 | 고객 문서 + 외부 데이터 자동 검색 |
| 전략 도출 | AI 기반 마케팅/콘텐츠 전략 자동 생성 |
| 실행 연동 | Strategy War Room → Hive Mind Studio |

## 4. Sources Panel 기능

| 기능 | 설명 |
|------|------|
| **Google Drive 연동** | OAuth `drive.file`, Google Picker |
| **Links 추가** | URL 입력, OG 메타데이터 추출 |
| **Notes 추가** | 마크다운 지원 텍스트 입력 |
| **토글 활성화** | 소스별 ON/OFF, 영구 히스토리 |
| **AI Discovery** | 외부 데이터 자동 검색 (뉴스, 리포트, 경쟁사) |
